# iPhone 12 Phone Stand
A phone stand designed for an iPhone 12. Holds the phone at an angle for stable viewing, video calls, and hands-free use.

I made this project as I was constantly annoyed at having to stop what im working on and pick up my phone to see what notification i had just gotten, and most of the time it was some random junk. With this, i'm finally able to just take a quick glance to see the notifcation and I only need to pick it up and reply if it's an important notification.
